SisIYA is an open source (GPL) system monitoring tool. It is a simple yet very powerfull tool. With SisIYA you can monitor your Linux, UNIX, MacOS X, Windows, network and power switches, wireless access points, printers, UPS, and any SNMP enabled systems. SisIYA checks and monitors for FTP, HTTP, IMAP, POP3, SMTP etc services. Extending SisIYA with new checks is a very simple task. One need only to write a script or a program to check the status of something (system. service etc), then it is a trivial task to send this information to the SisIYA. Let computers do the job for us!
